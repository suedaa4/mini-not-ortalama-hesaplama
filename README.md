# 📊 Mini Grade Calculator (Not Hesaplayıcı)

Bu proje, saf **JavaScript** kullanılarak fonksiyonlara nesne parametresi geçmeyi ve `return` mekanizmasını anlamayı amaçlayan bir algoritma alıştırmasıdır.

## 🚀 Özellikler
* Öğrenci bilgilerini ve notlarını nesne içinde saklama.
* Fonksiyona nesne göndererek dinamik veri işleme.
* `return` ile hesaplanan ortalamayı başka bir fonksiyona aktarma.
